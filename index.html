Här är några av mina projekt

Berras Bio

En ASP.Net Razor Pages sida där användare kan boka biobiljetter till ett urval av filmer Användare kan registera sig och få admin behörigheter för att kunna lägga till nya filmer för användare att boka biljetter till.

Hamster War

ASP.Net Blazor server med ASP.Net Web API för att kunna ladda upp hamstrar som sedan ska "slåss" för en användarens röst till vem som är sötast. Två hamstrar blir slumpmässigt valda inför tävlning och vinnaren får + i wins medans förloraren får + i förlust.

Mifamilia

En pinterest klon där användare kan ladda upp samt kommentera bilder. Inloggning sker via google authentication och användare kan se vem som har laddat upp bilder och filtrera bilder genom olika kategorier. React med Javascript använt till frontend och Sanity använt som backend.
